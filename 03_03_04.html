03.03.04 Animationsopgaven

Vegansk indkøb er et interaktivt animeret spil, som er bygget op at HTML, CSS og Java Script. Animationerne er lavet med CSS keyframe animationer og der er implimenteret lydeffekter i spillet.

Under processen har jeg lavet et aktivitetsdiagram og State Machine diagram for at få et overblik over hvordan brugeren oplever spillet og et overblik over hvilke funktioner det skal kodes med Java Spript.
